Dill Pixel offers two physics plugins:

- **Snap Physics**: A lightweight, AABB and circle-based collision system inspired by Maddy Thorson's TowerFall physics
- **Matter Physics**: A full-featured physics engine (using Matter.js) for complex physics simulations

## Plugin Comparison

| Feature        | Snap Physics                                                                                         | Matter Physics                                                                                      |
| -------------- | ---------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------- |
| Best Use Cases | • Platformers & endless runners<br/>• AABB & circle collisions<br/>• Precise, deterministic movement | • Physics-based puzzles<br/>• Complex body interactions<br/>• Realistic gravity & force simulations |
| Performance    | Minimal overhead, deterministic                                                                      | More resource-intensive, feature-rich                                                               |
| Complexity     | Simple, straightforward API                                                                          | Advanced features, steeper learning curve                                                           |

<Aside type="caution">Only use one physics plugin at a time to avoid performance issues and potential conflicts.</Aside>

<Aside type="tip">
  For performance-critical projects requiring precise platformer-style physics, start with Snap Physics and only switch
  to Matter if you need its advanced features.
</Aside>
