---
title: Accessibility Overview
---

dill-pixel provides comprehensive accessibility features through three main systems:

## 1. Focus Management

[Focus Management](/fundamentals/accessibility/focus-management) provides keyboard navigation and visual feedback for interactive elements, with support for:

- Focus layers for managing interactive elements
- Customizable visual feedback
- Keyboard navigation support

## 2. Voice-over Support

The [Voice-over Plugin](/fundamentals/accessibility/voiceover) handles audio narration with:

- Queue management
- Multiple play modes
- Localization support
- Integration with captions

## 3. Captions

The [Captions Plugin](/fundamentals/accessibility/captions) provides synchronized subtitles with:

- Multi-language support
- Customizable styling
- Automatic sync with voice-over
- Performance-optimized rendering
